SQL query developed using the **DVD Rental** database to calculate each customer's total spending and identify customers whose spending is above the overall average.

### Concepts used
- CTE
- JOIN
- GROUP BY
- SUM and AVG
- Window Functions (OVER)
- ORDER BY

**Database:** PostgreSQL (DVD Rental)
